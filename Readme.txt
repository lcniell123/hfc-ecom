Process to run it using a Mac.

- Download the folder 
- Move folder to desktop
- Open terminal and type "cd"
- Drag folder into the terminal 
- Press enter
- In case you do not have npm install it - https://docs.npmjs.com/downloading-and-installing-node-js-and-npm
- Type "npm install" to install all packages  
- Type "npm start" to start the program
